# 色が変わったボタンをタップしていく反射神経ゲーム
10回タップするとタイマーが停止する．
より短いタイムを目指したり友達と競ったりして遊べる．  
少し変えて制限時間内により多く，より正確にタップしてハイスコアを目指すゲームにしても面白そう．
